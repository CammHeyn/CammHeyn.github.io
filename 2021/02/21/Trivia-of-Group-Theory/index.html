Thm 1.2 Coset Theorem: two left (right) cosets of a subgroup are either identical or disjoint.

Proof \(\forall g_1,g_2\in G\) with subgroup \(H\sub G\). If there is \(h_1,h_2\in H\) such that \(g_1 h_1=g_2 h_2\), we then have \(g_1 H=g_1h_1\cdot h_1^{-1}H=g_2h_2\cdot H=g_2 H\). Likewise for right cosets. q.e.d.

Def 1.7 Conjugacy (between elements)

If there exists \(g\in G\) such that \(gfg^{-1}=h\), then \(f\sim h\). This is an equivalence relation.

Def 1.8 (Conjugacy) Class

Properties 1) identity itself constitutes a class. 2) elements of Abelian group each constitutes a class. 3) elements of the same class have the same order (but NOT the other way around).

Thm 1.4 Number of elements in a conjugacy class must be factor of the order of the (finite) group.

Proof We consider the conjugacy class which \(g\in G\) is in. Discover first that the elements that commute with \(g\) forms a subgroup \[ H_g:=\{h|h\in G, hg=gh\} \] To elaborate, for \(h_1,h_2\in H_g\), there is \(h_1g=gh_1\), \(h_2g=gh_2\), thus \(h_1h_2g=h_1gh_2=gh_1h_2\), namely \(h_1h_2\in H_g\), which states closure. Consider \(h\in H_g\), \(hg=gh\), then there is also \(g=h^{-1}gh\), which leads to \(gh^{-1}=h^{-1}g\), thus inverse property is justified. \(H_g\) is indeed a subgroup.

Then consider the equipartition by this subgroup through construction of left cosets, namely \[ \{g_0H_g,g_1H_g,\cdots\} \] where \(g_0,g_1\cdots\in G\). For any \(h\in H_g\), consider \((g_i h)g(g_i h)^{-1}=g_i hgh^{-1}g_i^{-1}=g_igg_i^{-1}\). This states that for any \(f\) in a coset, \(fgf^{-1}\) generates the same element. Different cosets generates different element in this way since, if we have \(g_i\) and \(g_j\) generating different cosets but still with \(g_igg_i^{-1}=g_jgg_j^{-1}\), there is \((g_j^{-1}g_i)g=g(g_j^{-1}g_i)\), which gives \(g_j^{-1}g_i\in H_g\), which gives \(g_j^{-1}g_iH_g=H_g\), which leads to \(g_i H_g=g_j H_g\) and that leads to contradiction.

Thus from the above facts, there are \(|G|/|H_g|\) cosets which generate each a conjugate element to \(g\) including \(g\) itself. Thus the order of the class is \(|G|/|H_g|\) which divides \(|G|\).

Def 1.9 Conjugate Subgroups

If there exists an element \(g\in G\) such that \(K=gHg^{-1}\), we then say \(K\) and \(H\) are conjugate to each other

Def 1.10 Invariant (normal) subgroup

If any conjugate class of the subgroup's element stays within the subgroup, the subgroup is invariant.

Thm 1.5 If \(H\) is invariant subgroup, \(\forall g\in G\), \(\mbox{Ad}_g(\bull)\equiv g\bull g^{-1}\) is a 1-1 map on \(H\). Thus the definition of invariant subgroup can be written \(gHg^{-1}=H, \forall g\in G\).

Prop All subgroups in Abelian group are invariant.

Thm 1.6 For an invariant subgroup, its left and right cosets coincide and vice versa.

\((\forall g\in G,gHg^{-1}=H)\Leftrightarrow(\forall g\in G,gH=Hg)\)

Prop Product of elements from two (not the subgroup) cosets of the invariant subgroup lands in another coset

Proof Consider equipartition of the group by the invariant subgroup \(H\), namely \[ G=\{g_0H,g_1H,g_2H,\cdots\} \] For any \(h_\alpha,h_\beta\in H\), consider first \((g_0h_\alpha)(g_0h_\beta)=h_\alpha h_\beta\in g_0 H\) and \((g_0h_\alpha)(g_i h_\beta)=h_\alpha g_i h_\beta=g_i(g_i^{-1}h_\alpha g_i)h_\beta=g_ih_\alpha'h_\beta\in g_i H\) and \((g_ih_\alpha)(g_0h_\beta)=g_ih_\alpha h_\beta\in g_i H\), where \(i\ne 0\).

Then consider \(i,j\ne 0\), there is \((g_ih_\alpha)(g_jh_\beta)=g_ig_jg_j^{-1}h_\alpha g_jh_\beta=g_ig_jh_\alpha'h_\beta\), which could be in neither \(g_i H\) nor in \(g_j H\), but in \((g_ig_j)H\).

Def 1.11 Quotient Group \(G/H\)

Suppose \(G\) has invariant subgroup \(H\), we can then divide \(G=\{g_0H,g_1H,\ldots\}\), treat each part as a new element in the quotient group \(G/H\).

Def 1.12 Isomorphism \(G\simeq F\)

Isomorphic mapping \(\Phi:G\longrightarrow F\)

Def 1.13 Homomorphism \(G\sim F~(|G|\geq|F|)\)

Homomorphic mapping \(\Phi:G\longrightarrow F\)

Def 1.14 Kernel of homomorphism

If \(G\sim F\), we call \(\Phi^{-1}(e_{F})\) the kernel of homomorphism

Thm 1.7 Kernel Theorem: If \(G\sim F\), then (1) \(\ker{G}\) is invariant subgroup of \(G\); (2) quotient group \(G/\ker G\) is isomorphic to \(F\).

Proof (1) Subgroup: \(\forall k_1,k_2\in\ker G\) that are distinct, there is \(\Phi(k_1)=\Phi(k_2)=e_F\). By property of homomorphism, there is \(\Phi(k_1k_2)=\Phi(k_1)\Phi(k_2)=e_F\), thus \((k_1k_2)\in \ker G\), which guarantees closure. Then consider \(\forall k\in\ker G\), then \(\Phi(k^{-1})\Phi(k)=\Phi(k^{-1})=\Phi(e_G)=e_F\), thus \(k^{-1}\in\ker G\).

Invariance: \(\forall g\in G\), consider \(g(\ker G)g^{-1}\), \(\forall k\in\ker G\), \(\Phi(gkg^{-1})=\Phi(g)\Phi(g^{-1})=e_F\), thus \(gkg^{-1}\in\ker G\).

  1. Denote \(\ker G= H\), consider quotient group \(G/H=\{g_0H, g_1 H,\cdots\}\), then \(\Phi(g_i H)=\Phi(g_i)\Phi(H)=\Phi(g_i)\). Together with \(\Phi(g_iHg_jH)=\Phi(g_kH)=\Phi(g_k)\), we could have the isomorphic mapping by combining \[ \Psi:G/H\longrightarrow F;~g_iH\mapsto \Phi(g_i) \]

Def 1.15 Automorphic mapping \(v:G\longrightarrow G\)

Preserves multiplication \(v(g_ig_j)=v(g_i)v(g_j)\).

Def 1.16 Automorphic group \(A(G)\)

Define \(v_1\cdot v_2\equiv v_1\circ v_2\) acting on \(G\) elements. Thus closure is guaranteed. Associativity is obvious. Identity and inverse is superficial.

Def 1.17 Inner-automorphic mapping \(\mbox{Ad}_u:g\mapsto ugu^{-1}\), such maps form a group called the Inner-automorphic group \(I(G)\).

Prop \(I(G)\) is an invariant subgroup of \(A(G)\)

Being a subgroup, there is closure: \(\mbox{Ad}_f\circ\mbox{Ad}_h(g)=fhgh^{-1}f^{-1}=\mbox{Ad}_{(fh)}(g)\); inverse: \((\mbox{Ad}_f)^{-1}=\mbox{Ad}_{f^{-1}}\).

Being invariant, there is \[ \begin{align} v\circ\mbox{Ad}_{f}\circ v^{-1}(g)&=v(f\cdot v^{-1}(g)\cdot f^{-1}) \\&= v(f)\cdot g\cdot v(f^{-1}) \\&= \mbox{Ad}_{v(f)}(g) \end{align} \] namely, \(v\circ\mbox{Ad}_{f}\circ v^{-1}=\mbox{Ad}_{v(f)}\). Thus \(I(G)\) is invariant.

Def 1.18 Transform \(f:X\longrightarrow X\), a 1-1 mapping; Complete symmetric group (on \(X\)) \(S_X\); Transformation (symmetric) group: subgroup of \(S_X\)

Thm 1.8 Cayley Theorem: Group \(G\) is isomorphic to a subgroup of \(S_G\).

obvious by picking \(g:G\longrightarrow G\) with \(h\mapsto gh\).

Def 1.19 Equivalence

If \(G\) is a transformation group on \(X\), and for \(x,y\in X,\exist g\in G\) such that \(g(x)=y\), we say that \(x,y\) are equivalent, denoted as \(x\sim y\) (with reflexivity, transitivity, symmetry).

Def 1.20 Orbit

If \(G\) is a transformation group on \(X\), \(x\in X\), then all elements in \(X\) that is equivalent to \(x\) forms a set, and is call the orbit of \(x\) on \(G\).

Def 1.21 Invariant subset

\(G\) is a transformation group on \(X\), if there is a subset \(Y\sub X\) that satisfies \(g(Y)\sub Y\) for any \(g\), we call \(Y\) an invariant subset of \(G\) on \(X\).

Prop Every orbit in \(X\) is \(G\)-invariant, thus all the orbits and their unions are \(G\)-invariant; For any subset \(Y\sub X\), we can always find a subgroup \(H\) of \(G\), such that \(Y\) is \(H\)-invariant

Def 1.22 Isotropy subgroup

\(G\) is transformation group on \(X\), \(x\in X\), a subgroup \(G^x\) of \(G\) holds \(x\) invariant, namely \(G^x=\{h|h\in G,hx=x\}\) and it is called the isotropy subgroup of \(G\) on \(x\). It is indeed a subgroup since for

Closure: \(\forall g_1,g_2\in G^x\), \(g_1g_2x=x\), thus \(g_1g_2\in G^x\).

Inverse: \(\forall g\in G^x\), since \(gx=x\), there is \(x=g^{-1}x\), thus \(g^{-1}\in G^x\)

Thm 1.9 \(G^x\) is isotropy subgroup of \(G\) on \(x\in X\), then every left coset of \(G^x\) maps \(x\) towards a specific point \(y\) in \(G\), with different image points for different cosets, i.e. we can construct a 1-1 mapping between \(x\)'s orbit and left cosets of \(G^x\).

Proof Consider the equipartition of \(G\) by \(G^x\), namely \[ G=\{g_0G^x,g_1G^x,\cdots\} \] we thus have \(Gx=\{g_0x,g_1x,\cdots\}\).

Def 1.23 Direct Product Group

\(G_1\otimes G_2\equiv\{g_{\alpha\beta}|g_{\alpha\beta}\equiv(g_{1,\alpha},g_{2,\beta}),g_{1,\alpha}\in G_1,g_{2,\beta}\in G_2\}\)

Def 1.24 Direct Product Factorization

If a group \(G\) has two subgroups \(G_1,G_2\) and every element in \(G\) can be uniquely written as \(g_{\alpha\beta}=g_{1\alpha}g_{2\beta}=g_{2\beta}g_{1\alpha}\) , then \(G\) is a direct product of \(G_1\) and \(G_2\), and they are called a direct product factorization of \(G\)

Rmk It is not required that \(G_1\), \(G_2\) be Abelian, we only require the multiplication between these two groups commute. We can further see that in the case of direct product factorization, \(G_1\) and \(G_2\) would share only a single element \(e\), and they are both invariant subgroups.

Ex \(\mathbb{R}^3=\mathbb{R}^2\otimes\mathbb{R}\), \(\mathbf{Z}_6=\{e,a^3\}\otimes\{e,a^2,a^4\}\), but not for \(D_3\) with \(G_1=\{e,d,f\}\) and \(G_2=\{e,a\}\).

Def 1.25 Semi-Direct Product

For groups \(G_1\), \(G_2\), if there exist an automorphism group \(A(G_1)\) which \(G_2\) is homomorphic to, namely \(\forall g_{2\beta}\in G_2\), we can have a specific automophism \(v_{g_{2\beta}}\) (through homomorphism) on \(G_1\). With this, we construct new element \(g_{\alpha,\beta}=\langle g_{1\alpha}g_{2\beta}\rangle\) with multiplication rule \(g_{\alpha,\beta}g_{\alpha',\beta'}=\langle g_{1\alpha}g_{2\beta}\rangle\langle g_{1\alpha'}g_{2\beta'}\rangle=\langle g_{1\alpha}v_{g_{2\beta}}(g_{1\alpha'})g_{2\beta}g_{2\beta'}\rangle\). This forms the semi-direct product group \(G_1\otimes_sG_2\)

Prop \(G_1\) is an invariant subgroup, while \(G_2\) is generally not. If otherwise, the semi-direct product levels up to be the direct product

Ex \(D_3=\{e,d,f\}\otimes_s\{e,a\}\) group.

Def 2.5 Linear Transform \(A:V\longrightarrow V\)

This can be written given a basis \[ (\bold e_1', \bold e_2',\cdots,\bold e_n')=(A\bold e_1,A\bold e_2,\cdots,A\bold e_n)=(\bold e_1,\bold e_2,\cdots,\bold e_n)[A] \] where \([A]\) is the matrix under the \(\{\bold e\}\) basis, and the transformation of a vector can be written \[ \left(\begin{array}{c} x_1'\\x_2'\\\vdots\\x_n' \end{array}\right) =[A\bold x]=[A][\bold x] =[A]\left(\begin{array}{c} x_1\\x_2\\\vdots\\x_n \end{array}\right) \]

Def 2.6 Linear Transform Group

Define multiplication to be successive action, then all non-singular linear transforms in n-dim'l complex linear space \(V\) forms a group, it is called the n-dim'l complex general linear group \(GL(V,\C)\).

Its subgroup \(L(V,\C)\) is called a linear transform group on \(V\).

Def 2.7 Group Representation

For a group \(G\), if there is a homomorphism \(A\) from \(G\) to an n-dim'l linear transform group \(L(V,C)\), then \(A\) is called a linear representation of \(G\), \(V\) is the representation space, \(n\) is the dimensionality of the representation.

With \(A:G\longrightarrow L(V,\C)\) being a homomorphism, \(\forall g_\alpha\in G,A(g_\alpha)\in L\); \(\forall g_\alpha,g_\beta\in G,A(g_\alpha g_\beta)=A(g_\alpha)A(g_\beta)\). With obvious correspondence for identity and inverse elements.

For isomorphism case, the representation is faithful.

Ex Trivial representation; There are cases when the basis of the representation space are chosen to be functions, i.e. the representation space is a linear function space \(V=\{f|f~\text{is function with some property on a set }X\}\), we can construct a representation scheme in the following fashion. Suppose \(G\) is a transformation group on \(X\) and we are representing \(G\) with linear representation \(A\) that is a homomorphism from \(G\) to \(V\), which is a linear function space on \(X\). For succintness, we set \(X=\R^3\), while \(g_\alpha\in G\) transforms the space \(X\) 1-1 into itself. Consider representation given by \[ (A(g_\alpha)\Psi)|_{g_\alpha x}\equiv\Psi(x) \] ​ for any \(\Psi\in V\). This is indeed a homomorphism since \[ (A(g_\alpha g_\beta)\Psi)|_{g_\alpha g_\beta x}=\Psi(x) \] ​ while \[ (A(g_\alpha)A(g_\beta)\Psi)|_{g_\alpha g_\beta x}=(A(g_\beta)\Psi)|_{g_\alpha^{-1}(g_\alpha g_\beta x)}=(A(g_\beta)\Psi)|_{g_\beta x}=\Psi(x) \]

Def 2.8 Equivalent Representation

For \(A:G\longrightarrow L(V,\C)\), and a non-singular linear transformation \(P\) on \(V\), then \(P^{-1}AP\) also gives a representation. Such representation is said to be equivalent to \(A\).

Rmk Representation spaces can be different for equivalent representations as long as their representation matrices can be related in the above fashion. For two representations to be equivalent, they must possess the same dimensionality. c.f. characteristics.

Def 2.9 Reducible Representation

If \(A\) is representation of \(G\) on \(V\), and there is a \(G\)-invariant non-trivial subspace \(W\), then we say \(A\) is reducible. This means that \(\forall y\in W\), \(\forall g_\alpha\in G\), \(A(g_\alpha)y\in W\). Under these conditions, we could easily construct a new representation with smaller dimensionality \[ A':G\longrightarrow L(W,\C) \] From this, we could naturally introduce the definition of an irreducible representation, namely if \(A\) is representation of \(G\) on \(V\), and there is no \(G\)-invariant non-trivial subspace so as to generate a new representation of smaller dimensionality, we call the representation irreducible.

Def 2.10 Direct Sum

Representation space \(V\) of \(G\) has subspace \(W\) and \(W'\). If \(\forall x\in V\), \(\exist!y\in W,z\in W',s.t.~x=y+z\), then we say \(V=W\oplus W'\).

This is guaranteed by \(W+W'=V\) and \(W\cap W'=\{0\}\).

c.f. Addition of linear space.

Def 2.11 Fully Reducible

If \(V=W\oplus W'\) and \(W\), \(W'\) are both \(G\)-invariant, then we say the representation is fully reducible. A fully reducible representation is of course reducible.

Fully reducible representation matrices may have the form \[ A(g_\alpha)= \left(\begin{array}{cc} C_\alpha&0\\0&B_\alpha \end{array}\right) \] We say that \(A\) can be reduced to be a direct sum of \(C\) and \(B\), denoted as \(A(g_\alpha)=C(g_\alpha)\oplus B(g_\alpha)\).

Eventually, ANY finite-dim'l representation can be reduced to (denoted as) \[ A(g_\alpha)=\sum_p\bigoplus_、{m_p}A^p(g_\alpha) \] where \(m_p\) is the repetition index of the irreducible representation \(A^p\).

Rmk If group \(G\) is represented by \(A\) on \(V\) and there is no \(G\)-invariant, non-trivial subspace, then \(A\) is said to be irreducible representation.

N.B. THE MAIN TASK OF GROUP REPRESENTATION THEORY IS TO FIND OUT ALL THE INEQUIVALENT AND IRREDUCIBLE REPRESENTATIONS OF ANY GROUP.

Thm 2.1 For finite group, if a representation is reducible, then it is fully reducible.

Proof The reducible representation can be written as \[ A(g_\alpha)= \left(\begin{array}{cc}G_1(g_\alpha)&R(g_\alpha)\\0&G_2(g_\alpha)\end{array}\right) \] Consider similar transformation of the form \[ P=\left(\begin{array}{cc}I&C\\0&I\end{array}\right) \] We claim that we could find a \(C\) such that \(P^{-1}A(g_\alpha)P=G_0(g_\alpha)\equiv\left(\begin{array}{cc}G_1(g_\alpha)&0\\0&G_2(g_\alpha)\end{array}\right)\). To show that this is so, consider equation \(A(g_\alpha)P=PG_0(g_\alpha)\), or more explicitly \[ \left(\begin{array}{cc} G_1(g_\alpha)&R(g_\alpha) \\0&G_2(g_\alpha) \end{array}\right) \left(\begin{array}{cc} I&C\\0&I \end{array}\right) = \left(\begin{array}{cc} I&C\\0&I \end{array}\right) \left(\begin{array}{cc} G_1(g_\alpha)&0 \\0&G_2(g_\alpha) \end{array}\right) \] which is \[ \left(\begin{array}{cc} G_1(g_\alpha)&G_1(g_\alpha)C+R(g_\alpha) \\0&G_2(g_\alpha) \end{array}\right) = \left(\begin{array}{cc} G_1(g_\alpha)&CG_2(g_\alpha) \\0&G_2(g_\alpha) \end{array}\right) \] and thus requires only \(G_1(g_\alpha)C+R(g_\alpha)=CG_2(g_\alpha)\).

From the fact that \(A\) is a representation \[ A(g_\alpha g)=\left(\begin{array}{cc}G_1(g_\alpha g)&R(g_\alpha g)\\0&G_2(g_\alpha g)\end{array}\right) \] equals \[ A(g_\alpha)A(g)= \left(\begin{array}{cc}G_1(g_\alpha)&R(g_\alpha)\\ 0&G_2(g_\alpha)\end{array}\right) \left(\begin{array}{cc}G_1(g)&R(g)\\ 0&G_2(g)\end{array}\right) = \left(\begin{array}{cc} G_1(g_\alpha)G_1(g)&G_1(g_\alpha)R(g)+R(g_\alpha)G_2(g) \\0&G_2(g_\alpha)G_2(g) \end{array}\right) \] Thus we have seen that \(G_1\) and \(G_2\) are separately representation of \(G\) and that \[ R(g_\alpha g)=G_1(g_\alpha)R(g)+R(g_\alpha)G_2(g) \] Thus the requirement of \(C\) can be rewritten as \[ G_1(g_\alpha)CG_2(g_\alpha^{-1})+R(g_\alpha)G_2(g_\alpha^{-1})=CG_2(g_\alpha)G_2(g_\alpha^{-1})=C \] Consider a direct construction \[ C=\frac{1}{|G|}\sum_{g\in G}R(g)G_2(g^{-1}) \] Thus \[ \begin{align} G_1(g_\alpha)CG_2(g_\alpha^{-1}) &=\frac{1}{|G|}\sum_{g\in G}G_1(g_\alpha)R(g)G_2(g^{-1})G_2(g_\alpha^{-1}) \\&=\frac{1}{|G|}\sum_{g\in G}G_1(g_\alpha)R(g_\alpha^{-1}g)G_2(g^{-1}) \\&=\frac{1}{|G|}\sum_{g\in G}G_1(g_\alpha)[G_1(g_\alpha^{-1})R(g)+R(g_\alpha^{-1})G_2(g)]G_2(g^{-1}) \\&=\frac{1}{|G|}\sum_{g\in G}[R(g)G_2(g^{-1})+G_1(g_\alpha)R(g_\alpha^{-1})] \\&=C+G_1(g_\alpha)R(g_\alpha^{-1}) \end{align} \] while \[ G_1(g_\alpha)R(g_\alpha^{-1})=R(g_0)-R(g_\alpha)G_2(g_\alpha^{-1})=-R(g_\alpha)G_2(g_\alpha^{-1}) \] where we have exploited that \(R(g_0)=0\) and thus, the requirement for \(C\) is reached.

Def 2.12 Unitary Representation

Representation that is unitary transformation group

Thm 2.2 If a unitary representation is reducible, then it is fully reducible

Proof \(A\) being a reducible unitary representation on \(V\), we can decompose \(V=W\oplus W^\perp\) where \(W\) is the \(G\)-invariant non-trivial subspace of \(V\). W.T.S. \(W^\perp\) is invariant as well.

For any \(g_\alpha\in G\) and \(y\in W\), there is \(A(g_\alpha)y\in W\). Thus for any \(z\in W^\perp\) there is \[ (A(g_\alpha)y|z)=(y|A(g_\alpha)^\dagger z)=(y|A(g_\alpha^{-1})z)=0 \] Namely, \(A(g_\alpha^{-1})z\in W^\perp\). Thus \(W^\perp\) is also \(G\)-invariant.

Rmk ANY finite-dim'l unitary representation can always be divided into direct sums of irreducible representations.

Def 2.13 Algebra

If \(R\) is linear space on field \(K\), and there is multiplication on \(R\) such that \(\forall x,y,z\in R, \forall\alpha\in K\)

  1. \(xy\in R\) (closure)

  2. \(x(y+z)=xy+xz\), \((x+y)z=xz+yz\) (associativity/compatibility with addition)

  3. \(\alpha(xy)=(\alpha x)y=x(\alpha y)\) (compatibility with number multiplication)

Then we say \(R\) is a linear algebra (or an algebra).

Rmk If we further require \((xy)z=x(yz)\), the algebra is associative algebra. And if we further require \(xy=yx\), the algebra is commutative algebra.

Def 2.14 Group Space

\(G\) is a group. If we further define linear combination of those elements to be vectors, by treating the \(g_\alpha\)'s as basis, on the field \(\C\). Then this linear space is called Group Space, \(V_G\).

We can see that group space is a constructed object out of thin air.

Def 2.15 Group Algebra

Define on \(V_G\) the multiplication \[ xy\equiv(\sum_\alpha x_\alpha g_\alpha)(\sum_\beta y_\beta g_\beta)=\sum_{\alpha,\beta}(x_\alpha y_\beta)(g_\alpha g_\beta) \] It is easily verified that (1) closure; (2) compatibility of vector multiplication with addition; (3) compatibility of vector multiplication with scalar multiplication are satisfied, so it is an algebra.

Further more, this algebra is associative. We call this algebra the group algebra \(R_G\).

Rmk With the definition of \(V_G\) and \(R_G\), we can generate a linear transformation on \(V_G\) from any group element \(g_i\) by the definition of \(R_G\), namely \[ L(g_i)x\equiv g_ix=\sum_j x_jg_ig_j \] Def 2.16 Left Canonical Representation

From the construction of the above linear transformation on \(V_G\), there is a homomorphism \(A:G\longrightarrow\{L(g_i)\}\) on which the multiplication of the \(\{L(g_i)\}\) are defined by successive action. The homomorphism is now a representation of the group.

We call this representation the left canonical representation.

Rmk Left canonical representation is furthermore a faithful representation (easily proven), not necessarily irreducible. Similarly we can define the right canonical representation \(R(g_i)x\equiv x g_i^{-1}=\sum_jx_jg_jg_i^{-1}\). And the right canonical representation is also a faithful representation of \(G\) with the proper definition of multiplication of \(\{R(g_i)\}\). Altogether they are called canonical representation or regular representation.

Thm 2.3 Schur's Lemma I

​ If group \(G\) has irreducible representation \(A\) and \(B\) on finite-dim'l vector spaces \(V_A\) and \(V_B\), respectively.

​ And if \(\exist\) linear transform \(M:V_A\longrightarrow V_B\), such that \(\forall g_\alpha\in G\), \(B(g_\alpha)M=MA(g_\alpha)\), then

  1. If \(A\) and \(B\) are inequivalent, \(M\equiv 0\).

  2. If \(M\ne 0\), \(A\) and \(B\) are equivalent. (Equivalent to (1))

Proof Check (1) only.

Take \(\ker(M)\sub V_A\), such subspace is \(G\)-invariant since \(\forall g_\alpha\in G\), \(\forall x\in\ker(M)\), \(MA(g_\alpha)x=B(g_\alpha)Mx=0\), thus \(A(g_\alpha)x\in\ker(M)\). But since \(A\) is irreducible, \(\ker(M)\) can only be trivial subspace of \(V_A\), otherwise \(\ker(M)\) would be a non-trivial, \(G\)-invariant subspace, rendering \(A\) reducible.

If \(\ker(M)=V_A\), \(M=0\) by definition.

If \(\ker(M)=0\), we can see that \(M\) must be injective (obvious), and surjective

(if not, \(M[V_A]\) would be a non-trivial, \(G\)-invariant subspace of \(V_B\), since \(B(g_\alpha)Mx=MA(g_\alpha)x\in M[V_A]\), and since \(B\) is irreducible, \(M[V_A]\) must be trivial, namely \(V_B\))

and thus bijective. Thus \(A(g_\alpha)=M^{-1}B(g_\alpha)M\), claiming equivalence, contradictory.

So \(\ker(M)=V_A\) and \(M\equiv 0\).

Thm 2.6 Orthogonality Theorem

If a finite group \(G\) has inequivalent, irreducible unitary representations \(A^1,A^2,\cdots,A^p,\cdots,A^r,\cdots\), with respective dimensionalities \(S_1,S_2,\cdots,S_p,\cdots,S_r,\cdots\). Proof We prove the first orthogonality first. Consider inequivalent, irreducible unitary representations \(A^p\) and \(A^r\). Construct a \(S_r\times S_p\) matrix \(C\) from any \(S_r\times S_p\) matrix \(D\), \[ C(D)\equiv\frac1n\sum_{i=1}^n A^r(g_i)DA^p(g_i^{-1}) \] We can easily see that \[ \begin{align} CA^p(g_j)&=\frac1n\sum_iA^r(g_i)DA^p(g_i^{-1})A^p(g_j) \\&= \frac{A^r(g_j)}{n}\sum_iA^r(g_j^{-1}g_i)DA^p((g_j^{-1}g_i)^{-1}) \\&= A^r(g_j)C \end{align} \] By Schur's Lemma I, we conclude that such \(C_{S_r\times S_p}(D)=0\). We substitute special \(D\), such that its elements are given by \[ D_{\mu_1\mu_2}=\delta_{\mu_1,\nu'}\delta_{\mu_2,\nu} \] Then elements of \(C\) is given by \[ \begin{align} C_{\mu'\mu}&=\frac1n\sum_{i=1}^n\sum_{\mu_1\mu_2}A^r_{\mu'\mu_1}(g_i)D_{\mu_1\mu_2}A^p_{\mu_2\mu}(g_i^{-1}) \\&= \frac1n\sum_{i=1}^nA^r_{\mu'\nu'}(g_i)A^p_{\nu\mu}(g_i^{-1}) \\&= \frac1n\sum_{i=1}^n[A^p_{\mu\nu}(g_i)]^*A^r_{\mu'\nu'}(g_i)=0 \end{align} \] The equation is true for any \((\mu\nu),(\mu'\nu')\), which gives the first orthogonality.

Now consider a similar construction generated by \[ C(D)\equiv \frac1n\sum_{i=1}^nA^p(g_i)DA^p(g_i^{-1}) \] And there is \[ CA^p(g_j)=A^p(g_j)C \] By Schur's Lemma II, we conclude that such \(C=\lambda(D)I_{S_p\times S_p}\). Define a set of basic functionals for \(V\) as \[ \bold e_{rj}:G\longrightarrow GL(V,\C), g_k\longmapsto\delta_{jk}e_r \] where \(j,k\in\{1,2,\cdots,n\}\) and \(r\in\{1,2,\cdots,d\}\) with \(e_r\) being a basis for \(W\). But as basis for the reduced \(V\), we pick only those with \(j\in\{1,2,\cdots,l\}\). Now we have \[ [U(g)\bold e_{rj}](g_s)\stackrel{\text{definition of }U}=\bold e_{rj}(g_s g)\stackrel{\text{coset repr.}}=\bold e_{rj}(hg_i)\stackrel{\text{restriction on }V}=B(h)\bold e_{rj}(g_i)=\delta_{ij}B(h)e_r \] where \(g_s\in G\) and we looked for unique \(h,g_i\) for every \(g_s\) as variable, since for every \(g_s g\), we could first pin down the coset index \(g_i\) where \(i\in\{1,\cdots,l\}\) and then \(h=g_s g g_i^{-1}\). Furthermore, there is symbolically \[ \bold e_{rj}(g_i)\equiv\bold e_{r(j+s-i)}(g_s) \] The transform relation now writes \[ [U(g)\bold e_{rj}](g_s)=B(h)\bold e_{rj}(g_i)=B(g_s g g_i^{-1})\bold e_{r(j+s-i)}(g_s) \] with attention to the restriction \(g_s g=hg_i\). The above is actually an equation of transformation relation componentwise, we could have as well extract the information contained by noting that the equation is nonzero iff \(i=j\) is met, meaning only those \(g_s\in\{g_s|g_sgg_j^{-1}\in H\}\) have nonzero components. We thus have, for those \(g_s\)'s, \[ U(g)\bold e_{rj}(g_s)=B(g_sg g_j^{-1})e_r \] while for the other \(g_s\)'s, we have \[ U(g)\bold e_{rj}(g_s)=0 \] Define \[ \dot{B}(h)= \left\{\begin{array}{} B(h)&h\in H\\ 0&h\notin H \end{array}\right. \] Thus we could write succinctly \[ U(g)\bold e_{rj}(g_s)=\dot{B}(g_s gg_j^{-1})e_r=\dot{B}(g_s gg_j^{-1})\bold e_{rs}(g_s)=[\sum_{t=1}^n\dot{B}(g_tgg_j^{-1})\bold e_{rt}](g_s) \] With the functionals outside of \(V\) given by \[ \bold e_{rt}(hg_\alpha)=B(h)\bold e_{rt}(g_\alpha)=0 \] if \(t>l\). Namely those \(\bold e_{rt}\) are not allowed in \(V\) by the restriction. Thus we have \[ U(g)\bold e_{rj}=\sum_{t=1}^l\dot{B}(g_t g g_j^{-1})\bold e_{rt} \] or in short, \[ U(g) \left(\begin{array}{c} \bold e_{11}&\bold e_{12}&\cdots&\bold e_{1l}\\ \bold e_{21}&\bold e_{22}&\cdots&\bold e_{2l}\\ \vdots&\vdots&\ddots&\vdots\\ \bold e_{d1}&\bold e_{d2}&\cdots&\bold e_{dl} \end{array}\right) = \left(\begin{array}{c} \bold e_{11}&\bold e_{12}&\cdots&\bold e_{1l}\\ \bold e_{21}&\bold e_{22}&\cdots&\bold e_{2l}\\ \vdots&\vdots&\ddots&\vdots\\ \bold e_{d1}&\bold e_{d2}&\cdots&\bold e_{dl} \end{array}\right) \left(\begin{array}{c} \dot{B}(g_1gg_1^{-1})&\dot{B}(g_1gg_2^{-1})&\cdots&\dot{B}(g_1gg_l^{-1})\\ \dot{B}(g_2gg_1^{-1})&\dot{B}(g_2gg_2^{-1})&\cdots&\dot{B}(g_2gg_l^{-1})\\ \vdots&\vdots&\ddots&\vdots\\ \dot{B}(g_lgg_1^{-1})&\dot{B}(g_lgg_2^{-1})&\cdots&\dot{B}(g_lgg_l^{-1}) \end{array}\right) \] where we symbolically decided that the \(\dot B\) should be acting on the \(\bold e\) from the left in the notational matrix multiplication. We should note that the matrix \(\dot{B}(g_t g g_j^{-1})\) is not that monstrous, since for \(g_t gg_j^{-1}\) to be in \(H\) is a strict case. For each \(g_t\), we could uniquely decompose \(g_tg=h_tg_t'\) and for \(h_t g_t'g_j^{-1}\) to be in \(H\) would require \(g_t'=g_j\), because if we have \(g_t'\ne g_j\) while \(h_tg_t'=h_t'g_j\), there is contradiction. Thus the matrix \(\dot{B}(g_t g g_j^{-1})\) has only one nontrivial entry per row. And likewise, there would only be one nontrivial entry per column, since for \(g_tg g_j^{-1}\) to be within \(H\) is to require that \(g_j g^{-1}g_t^{-1}\in H\). And for every \(g_j\) there is only one unique \(g_t\) for the job. Thus the transformation is de facto simple, having the form \[ U(g)\bold e_{rj}=B(g_{(\text{some entry decided by }j\text{ ofc lmfao) }}g g_j^{-1})\bold e_{r(\text{the same dull entry})} \] Thus the actual basis transformation would look like, with the basis listed in lexicographical order, \[ U(g)(\bold e_{11},\cdots,\bold e_{1l},\bold e_{21},\cdots,\bold e_{d1},\cdots,\bold e_{dl}) = (\bold e_{11},\cdots,\bold e_{dl}) \left(\begin{array}{c} \dot B_{l\times l}&~&~&~\\ ~&\dot B_{l\times l}&~&~\\ ~&~&\ddots&~\\ ~&~&~&\dot B_{l\times l} \end{array}\right)_{(dl)\times(dl)} \] where \[ \dot B_{l\times l}\equiv\left(\begin{array}{c} \dot{B}(g_1gg_1^{-1})&\dot{B}(g_1gg_2^{-1})&\cdots&\dot{B}(g_1gg_l^{-1})\\ \dot{B}(g_2gg_1^{-1})&\dot{B}(g_2gg_2^{-1})&\cdots&\dot{B}(g_2gg_l^{-1})\\ \vdots&\vdots&\ddots&\vdots\\ \dot{B}(g_lgg_1^{-1})&\dot{B}(g_lgg_2^{-1})&\cdots&\dot{B}(g_lgg_l^{-1}) \end{array}\right)=[B(g_igg_j^{-1})\delta_{i,\sigma(j)}] \] where \(\sigma\) is some permutation of \(\{1,\cdots,l\}\) and the \(i,j\) stand for some row and column indices.

Moreover, the matrix \(\dot{B}_{(dl)\times (dl)}\) may look as if it is block diagonal if we divide the matrix to \(l\times l\) blocks. But keep in mind that the \(B\)'s are also maps, so \(B\bold e\) would generally yield a linear combination of \(\bold e\)'s. If we look at some \(l\times l\) block directly, there is, as an example \[ U(g)(\bold e_{r1},\cdots\bold e_{rl})=(\bold e_{r1},\cdots,\bold e_{rl}) (B_{ij}\delta_{i,\sigma(j)})_{l\times l} \] Typically there is \[ U(g)\bold e_{rj}=B(g_{\sigma(j)}g g_j^{-1})\bold e_{r,\sigma(j)}=\sum_{t=1}^d b_{tr}(j)\bold e_{t,\sigma(j)} \] as a linear combination, where there is \[ B(h)e_r=\sum_{t=1}^de_tb_{tr}(h) \] since \(B(h)\) is a linear transformation taking a vector in \(W\) like \(e_r\) to combinations depending on \(B(h)\) itself. So genearally the map \(B\) in some column would distribute its numeric-representation along the column entries with a fix vertical spacing, namely once every \(l\) rows. So generally the transformation looks like \[ U(g)(\bold e_{r1},\cdots\bold e_{rl})=(\bold e_{r1},\cdots,\bold e_{rl}) (\mathcal{B}_{ij})_{l\times l} \] where \(\mathcal{B}_{ij}\) are numbers and the matrix \([\mathcal{B}_{ij}]\) is generally not block-diagonal. It can further be seen that \[ \dot{B}_{(dl)\times (dl)}=\sum_{j=1}^l [B(g_{\sigma(j)}g g_j^{-1}) ]^e_{d\times d}\otimes[E_{\sigma(j),j}]_{l\times l} \] where \(E_{ij}\) stands for the standard matrix with only the \((ij)\)-entry being 1 and other entries being set to zero.

The characteristics of induced representation is \[ \begin{align} \chi^U(g)&=\sum_{j=1}^l\mbox{tr}[B(g_{\sigma(j)}gg_j^{-1})]\cdot\mbox{tr}[E_{\sigma(j),j}] \\&=\sum_{j=1}^l\mbox{tr}[B(g_{\sigma(j)}gg_j^{-1})]\cdot\delta_{j,\sigma(j)} \\&=\sum_{j=1}^l\mbox{tr}[\dot{B}(g_{j}gg_j^{-1})]^e \end{align} \] By noting that \(\forall h\in H\), \(g_jgg_j^{-1}\in H\Leftrightarrow hg_jgg_j^{-1}h^{-1}\in H\), and \(g_j g g_j^{-1}\notin H\Leftrightarrow hg_jgg_j^{-1}h^{-1}\notin H\), we can further write \[ \chi^U(g)=\frac1m \sum_{j=1}^l\sum_{h\in H}\mbox{tr}\dot B(hg_jgg_j^{-1}h)=\frac1m\sum_{t\in G}\mbox{tr}\dot{B}(tgt^{-1}) \] The above matrix expansion is viewed from taking lexicographical order of the basis. If we reverse the role of \(r\lrarr j\), we would yield a simpler form as shown in the textbook.

Restriction of representation

If \(A\) is a representation of \(G\), then \(A\) is a representation of a subgroup \(H\), but denoted as \(A|_H\).

Rmk The restriction of an irreducible representation is not necessarily irreducible any longer. For example, restrict the 2-dim'l irreducible representation of \(D_3\) to the subgroup \(\{e,d,f\}\). c.f. von Neumann entropy for a composite pure state

Frobenius Theorem

If group \(G\) has subgroup \(H\), with \(A\) being an irreducible representation of \(G\) and \(B\) an irreducible representation on \(H\). We denote the induced representation by \(U\) and the restriction of \(A\) by \(A|_H\). There is \[ (\chi^A|\chi^U)=(\chi^B|\chi^{A|_H}) \] or namely, the repetition index of \(A\) in \(U\) is the same as the repetition index of \(B\) in \(A|_H\).

Proof There is \[ \begin{align} (\chi^A|\chi^U)&=\frac1n\sum_{g\in G}\chi^{A*}(g)\chi^U(g)=\frac1n\sum_{g\in G}\chi^{A*}(g)[\frac1m\sum_{t\in G}\mbox{tr}\dot{B}(tgt^{-1})] \\&=\frac{1}{nm}\sum_{g,t\in G}\chi^{A*}(g)\cdot\mbox{tr}\dot{B}(tgt^{-1}) \\&=\frac{1}{nm}\sum_{s,t\in G}\chi^{A*}(t^{-1}st)\cdot\mbox{tr}\dot{B}(s) \\&=\frac{1}{m}\sum_{s\in H}[\frac1n\sum_{t\in G}\chi^{A*}(s)]\cdot\mbox{tr}B(s) \\&=\frac1m\sum_{s\in H}\chi^{A*}(s)\chi^B(s)=(\chi^{A|_H}|\chi^B) \end{align} \] And since the repetition numbers are real, the desired equality is reached.
